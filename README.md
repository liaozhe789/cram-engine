# 🚀 期末速成引擎 (Cram Engine)

基于 **SM-2 间隔复习** + **知识图谱拓扑排序** 的 AI 考前速成系统。

## 它能做什么

- 📥 摄入往年真题、教材重点、老师题目（PDF / 图片 / 文本），自动构建知识图谱
- 🧠 四阶段闭环：**诊断 → 讲授 → 检题 → 补漏**，硬性状态机不跳阶段、不逃必考点
- 🔄 SM-2 间隔复习：考完不丢，自动调度复习时机
- 🎯 六种题型全覆盖：选择 / 判断 / 简答 / 案例分析 / 编程 / 公式推导
- 🪝 三种记忆钩子：口诀、对比表、荒诞锚定

## 快速开始

### 1. 加载课程
```
cram load
```
按引导填入课程名、教材、知识点列表、must_know 编号。

### 2. 摄入资料
把真题 PDF、老师 PPT、打印店题库扔进去，自动提取文本。

### 3. 开始学习
```
cram next
```
系统自动调度下一步——诊断测试 → 知识点讲授 → 检题 → 补漏。

### 4. 查看进度
```
cram status
```

## 文件结构
```
cram-engine/
  SKILL.md              # Codex skill 主控
  configs/              # 课程配置（YAML）
  knowledge/            # 知识图谱 + 摄入资料
  progress/             # 学习进度（含 SM-2 状态）
  scripts/              # Python 核心脚本
  templates/            # 6 个阶段教学模板
```
