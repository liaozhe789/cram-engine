1 查询选修了数据库原理的学生姓名。
2 查询比计算机系所有学生年龄都大的学生信息。
3列出“张力”选修的所有课程的课名和成绩。

学生表

选课表

1.查询选修了数据库原理的学生姓名。
连接查询方式：
SELECT Student.姓名
FROM Student, SC, Course
WHERE Student.学号 = SC.学号 
  AND SC.课号 = Course.课号 
  AND Course.课名 = N'数据库原理';
嵌套查询方式：
SELECT 姓名 
FROM Student 
WHERE 学号 IN (
    SELECT 学号 
    FROM SC 
    WHERE 课号 IN (
        SELECT 课号 
        FROM Course 
        WHERE 课名 = N'数据库原理'
    )
);

2.查询比计算机系所有学生年龄都大的学生信息。
SELECT * FROM Student
WHERE 年龄 > ALL (
    SELECT 年龄 
    FROM Student 
    WHERE 系号 = 'CS'
);
