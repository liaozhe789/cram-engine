> **考试概况**：单选 + 填空 + 计算 + 证明，四类题型。重心在 **第1–5章**，第6–7章仅涉及 2–3 道选择题（不出大题）。

## 第1章 整数的可除性 ★★★★☆

### 核心概念

| 概念 | 要点 |
|------|------|
| 整除 | $a \mid b \iff \exists c,\; b=ac$ |
| 带余除法 | $a = bq + r$，$0 \le r < b$，$q$ 和 $r$ **唯一** |
| 最大公因数 | $d = \gcd(a,b)$，辗转相除法 |
| 贝祖等式 | $\exists x,y \in \mathbb{Z},\; ax+by=\gcd(a,b)$ |
| 最小公倍数 | $\operatorname{lcm}(a,b) = \frac{|ab|}{\gcd(a,b)}$ |
| 互素 | $\gcd(a,b)=1$ |
| 算术基本定理 | 任意 $n>1$ 唯一分解为素数幂乘积 |

### 必会操作

- **辗转相除法** 求 $\gcd$（试卷A-1、作业题）
- **扩展辗转相除法** 求贝祖系数（试卷A-2、试卷B-3）
- **标准分解式**（试卷A-7、试卷B-7）

### 关键性质

1. $\gcd(a,b) = \gcd(b, a\bmod b)$
2. $\gcd(ma,mb) = m\cdot\gcd(a,b)$
3. $\gcd(a,b)=1$ 且 $a\mid bc \;\Rightarrow\; a\mid c$
4. $\gcd(a,b)=1$ 且 $a\mid c,\; b\mid c \;\Rightarrow\; ab\mid c$

### 常考证明

- $\gcd(ab,c)=\gcd(b,c)$ 当 $\gcd(a,c)=1$（**A卷+B卷都考**）
- $a^n-1$ 是素数 $\Rightarrow$ $a=2$ 且 $n$ 是素数（作业题）
- 带余除法中 $q,r$ 的唯一性

### 对应作业题

- 判断 101 是否为素数
- 求 $\gcd(144,84)$ 和 $\operatorname{lcm}(144,84)$
- 求 $72a+108b+96c$ 所能表示的最小正整数

## 第2章 同余 ★★★★☆

### 核心概念

| 概念 | 要点 |
|------|------|
| 同余 | $a \equiv b \pmod{n} \iff n \mid (a-b)$ |
| 完全剩余系 | 模 $m$ 下恰好覆盖 $\{0,1,\dots,m-1\}$ 各一次 |
| 简化剩余系 | 完全剩余系中与 $m$ 互素的那些类 |
| 欧拉函数 | $\varphi(n)=n\prod_{p\mid n}(1-\frac{1}{p})$ |
| 欧拉定理 | $\gcd(a,n)=1 \;\Rightarrow\; a^{\varphi(n)}\equiv 1 \pmod{n}$ |
| 费马小定理 | $p$ 素数 $\;\Rightarrow\; a^{p}\equiv a \pmod{p}$ |
| 模幂运算 | 平方-乘算法 (square-and-multiply) |

### 必会公式

- $\varphi(p)=p-1$（$p$ 素数）
- $\varphi(p^k)=p^k-p^{k-1}$
- $\varphi(mn)=\varphi(m)\varphi(n)$ 当 $\gcd(m,n)=1$
- $a^{\varphi(n)}\equiv 1 \pmod{n}$（$\gcd(a,n)=1$）

### 常考题型

- **完全剩余系/简化剩余系** 判定与列举（A卷填空1-2、B卷填空1-2，A卷选择4、B卷选择6）
- **欧拉函数** 计算（A卷填空5、B卷填空5）
- **模幂运算**：$2^k \bmod 7$ 类型，用于“星期几”问题（A卷填空4、B卷填空4）
- **费马小定理** 应用：求 $a^p \bmod p$

### 模幂运算（平方-乘算法）

> 计算 $a^k \bmod n$：将 $k$ 写为二进制，从高位到低位扫描，遇 1 则乘。

**例题**：求 $2^{50} \bmod 7$。

$50 = (110010)_2$。$2^1\equiv 2$，$2^2\equiv 4$，$2^3\equiv 1$，模 7 周期为 3。
$50 \bmod 3 = 2$，故 $2^{50}\equiv 2^2\equiv 4 \pmod{7}$。

### 常考证明

- 费马小定理：$a^p\equiv a \pmod{p}$（**B卷证明**）
- 欧拉定理：$a^{\varphi(m)}\equiv 1 \pmod{m}$（**B卷证明**）
- $a^7 \equiv a \pmod{63}$ 当 $\gcd(a,3)=1$（**A卷证明**）

### 对应作业题

- 求 $2^{50}$ 除以 7 的余数

## 第3章 一次同余方程 ★★★★★

> **最重要章节之一**，RSA 即基于此。

### 核心概念

| 概念 | 要点 |
|------|------|
| 一次同余方程 | $ax \equiv b \pmod{m}$，有解 $\iff \gcd(a,m) \mid b$ |
| 逆元 | $a^{-1} \bmod m$ 存在 $\iff \gcd(a,m)=1$，用扩展辗转相除法求 |
| 中国剩余定理 (CRT) | 模数两两互素的同余方程组有唯一解（模乘积意义下） |
| RSA | 基于大整数分解困难性的公钥密码 |

### 一次同余方程求解步骤

$ax \equiv b \pmod{m}$，$d=\gcd(a,m)$：

1. 若 $d \nmid b$，无解
2. 若 $d \mid b$，两边除以 $d$，化为 $a''x \equiv b'' \pmod{m'}$，其中 $\gcd(a',m')=1$
3. 求 $a'$ 模 $m'$ 的逆元，两边同乘即得

### 中国剩余定理 (CRT) 标准步骤

对于 $\begin{cases} x\equiv a_1 \pmod{m_1} \\ x\equiv a_2 \pmod{m_2} \\ \cdots \end{cases}$：

1. $M = m_1 m_2 \cdots m_k$
2. $M_i = M/m_i$
3. 求 $y_i \equiv M_i^{-1} \pmod{m_i}$
4. $x \equiv \sum a_i M_i y_i \pmod{M}$

### RSA 核心公式

- $n=pq$，$\varphi(n)=(p-1)(q-1)$
- 选 $e$ 使 $\gcd(e,\varphi(n))=1$
- 求 $d \equiv e^{-1} \pmod{\varphi(n)}$（扩展辗转相除法）
- 公钥 $(n,e)$，私钥 $d$
- 加密：$c \equiv m^e \pmod{n}$
- 解密：$m \equiv c^d \pmod{n}$

**老师强调的考点**：欧拉函数条件、求逆元、RSA 全套流程

### 常考证明

- **RSA 解密正确性**（**A卷证明**）：需证 $m^{ed}\equiv m \pmod{n}$，分 $\gcd(m,n)=1$ 和 $\gcd(m,n)\neq 1$ 两种情况

### 对应作业题

- 解同余方程 $7x \equiv 4 \pmod{13}$
- CRT：$\begin{cases} x\equiv 3\pmod{5} \\ x\equiv 4\pmod{7} \end{cases}$
- 已知 $\varphi(n)=120$，$e=17$：(1) 扩展辗转相除法求 $\gcd(120,17)$ 及贝祖系数；(2) 求 RSA 私钥指数 $d$

## 第4章 二次同余 ★★★★☆

### 核心概念

| 概念 | 要点 |
|------|------|
| 二次剩余 | $x^2\equiv a \pmod{p}$ 有解时 $a$ 为模 $p$ 的二次剩余 |
| 勒让德符号 | $\left(\frac{a}{p}\right)$：有解为 $1$，无解为 $-1$，$p\mid a$ 为 $0$ |
| 欧拉判别法 | $\left(\frac{a}{p}\right)\equiv a^{\frac{p-1}{2}} \pmod{p}$ |
| 二次互反律 | $\left(\frac{p}{q}\right)\left(\frac{q}{p}\right)=(-1)^{\frac{p-1}{2}\cdot\frac{q-1}{2}}$ |

### 必会公式

- $\left(\frac{-1}{p}\right)=(-1)^{\frac{p-1}{2}}$：$p\equiv 1\pmod{4}$ 时 $-1$ 是二次剩余
- $\left(\frac{2}{p}\right)=(-1)^{\frac{p^2-1}{8}}$
- 勒让德符号是积性函数：$\left(\frac{ab}{p}\right)=\left(\frac{a}{p}\right)\left(\frac{b}{p}\right)$
- 非零二次剩余恰有 $\frac{p-1}{2}$ 个

### 解的个数判断

$x^2 \equiv a \pmod{n}$，其中 $n=p_1^{e_1}\cdots p_k^{e_k}$：
- 先判断每个 $p_i$ 下是否有解（用勒让德符号/欧拉判别法）
- 每个素因子贡献 2 个解（$p_i\neq 2$ 且 $a\not\equiv 0$）
- CRT 组合得 $2^k$ 个解

### 常考题型

- **判定二次同余是否有解及解的个数**（A卷选择6、B卷选择4）
- **勒让德符号** 计算（B卷填空3、作业题）
- **求解二次同余式**（B卷计算2）

### 对应作业题

- 判断 $x^2\equiv 56 \pmod{107}$ 是否有解
- 计算勒让德符号 $\left(\frac{151}{373}\right)$
- 判断方程 $11x^2\equiv -3 \pmod{91}$

## 第5章 原根和离散对数 ★★★★☆

### 核心概念

| 概念 | 要点 |
|------|------|
| 阶 | 使 $a^k\equiv 1\pmod{m}$ 的最小正整数 $k$，记作 $\operatorname{ord}_m(a)$ |
| 原根 | $\operatorname{ord}_m(g)=\varphi(m)$ 时 $g$ 为模 $m$ 的原根 |
| 离散对数 | $\operatorname{ind}_g(a)$：使 $g^k\equiv a\pmod{m}$ 的最小非负 $k$ |
| ElGamal | 基于离散对数困难问题的公钥密码 |

### 关键性质

- $\operatorname{ord}_m(a) \mid \varphi(m)$
- $a^k\equiv 1 \pmod{m} \iff \operatorname{ord}_m(a) \mid k$
- 模 $p$ 的原根个数：$\varphi(p-1)$
- $g$ 是原根 $\iff$ 对 $\varphi(m)$ 的每个素因子 $q$，$g^{\varphi(m)/q}\not\equiv 1 \pmod{m}$

### 求所有原根步骤（模素数 $p$）

1. 计算 $\varphi(p)=p-1$，分解 $p-1$ 的素因子
2. 找到第一个原根 $g$（逐一测试，$g$ 从 2 开始）
3. 所有原根：$\{g^k \mid 1\le k\le p-1,\; \gcd(k,p-1)=1\}$

### 离散对数的性质

- $\operatorname{ind}_g(ab)\equiv \operatorname{ind}_g(a)+\operatorname{ind}_g(b) \pmod{\varphi(m)}$
- $\operatorname{ind}_g(a^k)\equiv k\cdot\operatorname{ind}_g(a) \pmod{\varphi(m)}$

### 常考题型

- **求模 $m$ 的原根**（B卷选择10、B卷计算3）
- **阶的证明**（老师口头强调）
- **离散对数基本性质**（老师口头强调）

## 第6章 近世代数基础 ★☆☆☆☆

> 仅 1–2 道选择题，掌握基本定义即可。

### 只需记住

| 结构 | 一句话定义 |
|------|-----------|
| 群 | 集合 + 封闭、结合、有单位元、有逆元的二元运算 |
| 阿贝尔群 | 满足交换律的群 |
| 循环群 | 由一个元素生成的群 |
| 环 | 有加法（阿贝尔群）+ 乘法（满足结合律和分配律） |
| 域 | 非零元都有乘法逆元的交换环 |
| 有限域 | $\operatorname{GF}(p^n)$，阶为 $p^n$ |

### 必记结论

- 子群的阶整除群的阶（拉格朗日定理）
- 元素的阶整除群的阶
- $\mathbb{Z}_p$ 是域 $\iff$ $p$ 是素数
- $\mathbb{F}_p^*$ 是 $p-1$ 阶循环群

## 复习优先级建议

1. **最高优先**：扩展辗转相除法、CRT、RSA 全套（加密/解密/私钥计算/正确性证明）
2. **次高优先**：欧拉函数、费马小定理、二次剩余判定（勒让德符号）、求原根
3. **中等优先**：完全/简化剩余系、模幂运算（星期几问题）、标准分解式
4. **低优先**：第6章群/环/域定义（1–2道选择）、第7章椭圆曲线基本概念（1道选择，不出大题）
